# descriptiveStats

An R package providing functions for common descriptive statistics: mean, median, mode, Q1, Q3, and IQR.

## Installation

```r
devtools::install("question_1/descriptive_stats")
library(descriptiveStats)
```

## Functions

| Function | Description |
|---|---|
| `calc_mean(x)` | Arithmetic mean |
| `calc_median(x)` | Median |
| `calc_mode(x)` | Mode (handles ties and no-mode cases) |
| `calc_q1(x)` | First quartile (25th percentile, type 7) |
| `calc_q3(x)` | Third quartile (75th percentile, type 7) |
| `calc_iqr(x)` | Interquartile range (Q3 - Q1) |

All functions accept a numeric vector and silently remove `NA` values before computation.

## Usage

```r
data <- c(1, 2, 2, 3, 4, 5, 5, 5, 6, 10)

calc_mean(data)    # 4.3
calc_median(data)  # 4.5
calc_mode(data)    # 5
calc_q1(data)      # 2.25
calc_q3(data)      # 5
calc_iqr(data)     # 2.75
```

## Edge-Case Handling

- **Empty vectors / all-NA vectors**: raise an informative error.
- **Non-numeric input** (character, logical): raises an error indicating the expected type.
- **Mode ties**: `calc_mode()` returns all values sharing the highest frequency.
- **No unique mode**: `calc_mode()` returns `NA` with a message when all values occur equally.
- **Single value**: all functions work; `calc_iqr()` returns 0.
